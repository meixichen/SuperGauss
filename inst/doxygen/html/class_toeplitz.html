Detailed Description

Class for Toeplitz-related computation

Generate the memory for computation including multiplying, solving, determinant, first and second order derivative of the log-determinant:

  • Multiplication between Toeplitz matrix \(T \times x\) and vectors is obtained with circulant embeding.
  • Multiplication between inverse Toeplitz matrix and vector \(T^{-1} \times x\) involves the Gohberg-Semencul formula which decomposes inverse Toeplitz into sum of products of lower and upper triangular Toeplitz matrices: \( T^{-1} = \frac{1}{\rho_0} [L_1'L_1 - L_2' L_2] \), where \(L_1, L_2\) are lower-trangular Toeplitz matrices and \(\rho_0\) is the first element of GSchur Algorithm output: length-N vector \(\rho\).
  • Determinant of Toeplitz matrix is the direct result of GSchur algorithm.
  • First order derivative of log-determinant:

    \[ \frac{\partial \log\mid T^{-1}\mid}{\partial \theta} = \text{tr} (T^{-1} V), \text{ where } V = \frac{\partial T}{\partial \theta} \]

    V is also Toeplitz since it is the derivative of Toeplitz matrix, and has displacement rank 2 : \(V = \frac{1}{t_0} [U_1 U_1' - U_2 U_2']\), where \(U_1\) is a upper triangular Toeplitz matrix. Thus

    \begin{eqnarray*} \text{tr} (T^{-1} V) &=& \frac{1}{\rho_0 t_0} \text{tr} \{ [L_1'L_1 - L_2' L_2][U_1 U_1' - U_2 U_2'] \} \\ &=& \frac{1}{\rho_0 t_0} (\text{tr} \{ U_1'L_1 L_1' U_1 \} + ...) \\ &=& \frac{1}{\rho_0 t_0} (\text{tr} \{ A_1 A_1' \} + ...) \end{eqnarray*}

    where \(A = U_1'L_1\) is the product of two lower triangular Toeplitz matrices and is also a lower triangular Toeplitz matrix, making computing \(\text{tr} \{ A_1 A_1' \}\) efficient.
  • Second order derivative of log-determinant:

    \[ \frac{\partial^2 \log\mid T^{-1}\mid}{\partial \theta_1 \partial \theta_2} = \text{tr} (T^{-1} V_0) - \text{tr} (T^{-1} V_1 T^{-1} V_2), \text{ where } V_0 = \frac{\partial^2 T}{\partial \theta_1 \partial \theta_2}, V_1 = \frac{\partial T}{\partial \theta_1}, V_2 = \frac{\partial T}{\partial \theta_2} \]

    \(V_0, V_1, V_2\) are Toeplitz matrices. Computation of \(\text{tr} (T^{-1} V_1 T^{-1} V_2)\) is non-trival and details are in paper.

Trace function, returns \( \text{tr}(T^{-1} V_1 T^{-1} V_2)\).

Trace function for second derivative of log-determinant.

Computes \( \text{tr} (T^{-1} V_1 T^{-1} V_2) \), where \(V_1, V_2\) are two Toeplitz matrices.

If the first element of Toeplitz matrix \(V_2\): \(V_2[1,1]\) is very small(criteria is 0.0001) in this implementation, we use following steps to stabilize the algorithm:

\[ \text{tr}(T^{-1} V_1 T^{-1} V_2) = \text{tr}(T^{-1} V_1 T^{-1} (V_2+I)) - \text{tr}(T^{-1} V_1 T^{-1}) \]

where \(I\) is an identity matrix.

Parameters
[in]acf1Real vector of first column of Toeplitz matrix \(V_1\).
[in]acf2Real vector of first column of Toeplitz matrix \(V_2\).

Trace function, returns \( \text{tr}(T^{-1} V)\).

Trace function for derivative of log-determinant.

Computes \( \text{tr} (T^{-1} V) \), where \(V\) is a Toeplitz matrix.

If the first element of Toeplitz matrix \(V\): \(V[1,1]\) is very small(criteria is 0.0001) in this implementation, we use following steps to stabilize the algorithm:

\[ \text{tr}(T^{-1} V) = \text{tr}(T^{-1} (V+I)) - \text{tr}(T^{-1}) \]

where \(I\) is an identity matrix.

Parameters
[in]acf0Real vector of first column of Toeplitz matrix \(V\).
