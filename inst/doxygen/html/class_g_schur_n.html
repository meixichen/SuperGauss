Detailed Description

Generalized Schur Algorithm for arbitrary size N.

Size of input is not necessarily \(2^K \times b\).

The implementation of Generalized Schur algorithm for any N contains following steps:

1) Decompose integer N into the binary form w.r.t modulus b and obtain a vector s s.t. \(s = {2^{k_0} \times b, 2^{k_1} \times b, ..., 2^{k_T} \times b, r}\), where \(k_0 > ... > k_T \geq 0\), \(r < b\), and sum(s) = x. This step is done with function int2Bin. As in GSchur Algorithm, whole size-N transformation \(T_{0,N}\) can be decomposed into binary pieces in following way

\[ T_{0,N} = T_{0, s[1]} \circ T_{s[1], s[2]} \circ T_{s[1]+s[2], s[2]} \circ ... \circ T_{s[1]+s[2]+...s[m], r} \]

2) Use doubling procedure to compute the binary pieces \(T_{s[1]+s[2]+..., s[i]}\). This step is done with function GenStep of GSchurN class.

3) Merge the binary pieces in ascending order (for computation efficiency) and finally obtain the size-N transformation \(T_{0,N}\). This step is done with function GenMerge of GSchurN class.
