Detailed Description

Class for Toeplitz-related computation

Generate the memory for computation including multiplying, solving, determinant, first and second order derivative of the log-determinant.

Multiplication between Toeplitz matrix and vectors is obtained with circulant embeding.

Multiplication between inverse Toeplitz matrix and vector involves the Gohberg-Semencul formula which decomposes inverse Toeplitz into sum of products of lower and upper triangular Toeplitz matrices: \( T^{-1} = \frac{1}{\rho_0} [L_1'L_1 - L_2' L_2] \), where \(L_1, L_2\) are lower-trangular Toeplitz matrices and \(\rho_0\) is the first element of GSchur Algorithm output vector \(\phi\).

Determinant of Toeplitz matrix is the direct result of GSchur algorithm.

First order derivative of log-determinant:

\[ \frac{\partial \log\mid T^{-1}\mid}{\partial \theta} = \text{tr} (T^{-1} \frac{\partial T}{\partial \theta}) \]
